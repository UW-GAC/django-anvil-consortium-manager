Adapter overview
================

This app provides the ability to customize behavior to suit the needs of the project.
Customization is generally done by defining custom adapters that can change or control app behavior.

The following sections provide more information about the different adapters and options for customizability.

- Accounts: :ref:`account_adapter`
- Workspaces: :ref:`workspace_adapter`
- ManagedGroups: :ref:`managed_group_adapter`
